# Keylogger

A simple keylogger that displays keyboard input in the terminal.

## Installation

1. Install the required dependency:
```bash
pip install -r requirements.txt
```

## Usage

Run the keylogger:
```bash
python keylogger.py
```

The program will:
- Display "Connection established" message
- Start collecting and displaying keyboard input horizontally in the terminal
- Continue until you press Ctrl+C to stop

## Note

This tool is for educational and personal use only. Make sure you have proper authorization before using keyloggers on any system.
